# FloatTodo

FloatTodo 是一个基于 AI 项目拆解的桌面悬浮学习任务管理系统，面向大学生课程作业、项目开发和日常学习管理场景。

## 主要功能

1. 普通待办管理：新增、删除、完成任务。
2. 任务优先级：紧急、重要、普通。
3. 截止时间提醒：显示截止时间和逾期状态。
4. 日常记录：喝水 +1，记录今日次数、上次喝水时间、距离上次喝水多久。
5. AI 项目拆解：输入项目描述，AI 自动拆成小任务。
6. 候选任务确认：用户选择 AI 生成的小任务加入待办。
7. 桌面悬浮窗：常驻桌面显示今日任务。
8. 桌宠提醒：通过桌宠状态和气泡提示反馈任务状态。

## 技术路线

- 开发语言：C#
- 框架：WPF
- 数据存储：本地 JSON
- 版本管理：GitHub
- AI 能力：大语言模型 API 或 Mock 模拟数据

## 项目目标

帮助学生把复杂项目拆解成可执行的小任务，同时管理日常学习和生活记录。
